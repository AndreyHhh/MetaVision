## 📋 О ПРОГРАММЕ

MetaVision for 1C — это профессиональный инструмент для анализа, аудита и визуализации конфигураций 1С:Предприятие.

**Программа помогает:**
- Понимать структуру сложных конфигураций
- Находить уязвимости и проблемы производительности
- Анализировать связи между функциями
- Документировать код

---

## 🚀 БЫСТРЫЙ СТАРТ

1. Нажмите **«Выбрать папку»** и укажите путь к конфигурации 1С
2. Нажмите **«Загрузка данных конфигурации»**
3. Дождитесь завершения (прогресс-бар покажет статус)
4. Выберите объект в дереве слева
5. Выберите функцию в таблице
6. Анализируйте граф и код справа

---

## 🎯 ВОЗМОЖНОСТИ

### 📊 Анализатор кода
- Дерево метаданных с навигацией
- Таблица функций с цветными индикаторами
- Граф структуры функции (3 стиля)
- Подсветка синтаксиса 1С

### 🛠️ Работа с модулем
- Преобразование запросов (с `|` и обратно)
- Валидация синтаксиса
- Проверка орфографии (Яндекс.Спеллер)
- Автоформатирование кода

### 🔍 Поиск по модулям
- Полнотекстовый поиск по всем функциям
- Контекст результатов
- Просмотр полного кода функции

### 🔒 Сканер проблем в коде
- Безопасность (RCE, SSRF, COM, инъекции)
- Многопоточность
- Плохие имена переменных
- Транзакции и блокировки
- Запросы в циклах

### ⚡ Анализатор производительности
- Поиск запросов в циклах
- Поиск вложенных элементов
- Визуализация цепочек вызовов

### 📈 Статистика
- Карточки с показателями
- Диаграммы распределения
- Топ-10 самых длинных/сложных функций

## 🔒 БЕЗОПАСНОСТЬ И КОНФИДЕНЦИАЛЬНОСТЬ

MetaVision работает **строго локально**. Исходный код вашей конфигурации:
- ❌ Не отправляется в интернет
- ❌ Не анализируется на сторонних серверах
- ❌ Не передаётся третьим лицам

Вы можете убедиться в этом лично, изучив исходный код в этом репозитории.
